## Descripción
Sistema orientado a objetos desarrollado en Python para la gestión de clientes, servicios y reservas de la empresa Software FJ.

El proyecto implementa:
- Abstracción
- Herencia
- Polimorfismo
- Encapsulación
- Manejo avanzado de excepciones
- Registro de logs
- Simulación de operaciones válidas e inválidas

## Ejecución
Ejecutar el archivo principal en Python:

```bash
Fase4.py
```

## Archivo de logs
